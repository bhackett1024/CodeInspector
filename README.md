# JIT Inspector #


## Build Instructions ##

From inside the project directory, run:

make

or:

zip -r ../JITInspector.xpi * -x ".git/*"


## Installation ##

Drag and drop JITInspector.xpi in Firefox, follow instructions to install add-on.
